# 打字练习小项目设计

## 目标

做一个适合朋友日常练习的网页打字工具。项目以“打开就能练”为核心，不做登录和复杂后台，优先把练习体验、视觉质感、统计反馈和部署便利性做好。

## 推荐方案

采用 Vite + React + TypeScript 构建静态单页应用。开发体验清晰，后续扩展方便；构建后产物是静态文件，可以部署到常见服务器或静态托管平台。

## 页面风格

整体做成“专注练习舱”的感觉：深色底、柔和高亮、清晰信息层级。主区域突出待输入文本和输入状态，辅助信息放在两侧和顶部，避免页面像普通表单。

视觉元素包括：

- 顶部模式栏：中文、英文、符号、代码、混合模式切换。
- 中央练习区：大字号文本逐字符高亮，当前字符有明确光标提示。
- 下方输入区：隐藏真实文本框的生硬感，做成专注输入面板。
- 右侧统计面板：WPM、准确率、用时、错误数、进度。
- 底部历史记录：展示最近几次成绩，帮助看到进步。

## 核心功能

1. 模式切换
   - 中文短句：常用短句、生活化表达。
   - 英文句子：单词、短句、大小写和标点。
   - 符号练习：括号、引号、冒号、分号等。
   - 代码片段：JavaScript/HTML/CSS 常见片段。
   - 混合练习：从以上素材中随机组合。

2. 实时输入反馈
   - 正确字符显示绿色或亮色。
   - 错误字符显示红色并轻微提示。
   - 当前字符显示光标或下划线。
   - 支持退格修正。

3. 练习统计
   - WPM：按英文标准折算，中文也按字符节奏统计。
   - 准确率：正确输入字符数 / 已输入字符数。
   - 错误数：累计错误击键。
   - 用时：从第一次输入开始计时。
   - 进度：当前字符 / 总字符。

4. 练习控制
   - 重新开始当前文本。
   - 下一段随机文本。
   - 选择练习时长：不限时、1 分钟、3 分钟。
   - 开启/关闭声音反馈，默认关闭。

5. 本地记录
   - 使用 localStorage 保存最近 8 次成绩。
   - 不需要账号，不上传数据。

## 交互流程

用户进入页面后默认看到混合模式的一段练习文本。点击练习区或直接键盘输入后开始计时。输入过程中页面实时高亮每个字符，并更新统计。完成一段后显示完成状态、成绩摘要和“再来一段”按钮，同时把结果写入本地历史记录。

## 错误与边界处理

- 未开始输入时 WPM 和准确率保持初始状态。
- 练习文本完成后忽略多余输入。
- 切换模式时自动刷新文本并重置统计。
- localStorage 不可用时不阻塞练习，只是不保存历史。
- 移动端可使用系统键盘，但主要体验针对桌面键盘练习。

## 测试重点

- 不同模式能生成不同练习文本。
- 正确、错误、退格、完成状态都能正确计算。
- 计时、WPM、准确率和历史记录稳定。
- 桌面和移动视口下文本不重叠，按钮不溢出。
